Licencias de software

son acuerdos legales que establecen los términos y condiciones bajo los cuales los usuarios pueden utilizar un programa de software. Estas licencias definen los derechos y restricciones que tienen los usuarios con respecto al software y son emitidas por el titular de los derechos de autor del software, que a menudo es la empresa o individuo que lo desarrolló. Las licencias de software son importantes porque regulan cómo se puede usar, copiar, distribuir y modificar el software.


Tipos de licencia de software

  1. Licencia de Software Propietario: En este tipo de licencia, el propietario del software retiene la mayoría de los derechos y control sobre el software. Los usuarios suelen obtener una licencia limitada para usar el software según los términos y condiciones establecidos por el propietario. Ejemplos de esto incluyen software comercial como Microsoft Office.
  2. Licencia de Código Abierto (Open Source): Estas licencias permiten a los usuarios ver, modificar y distribuir el código fuente del software. Algunas de las licencias de código abierto más conocidas incluyen la Licencia Pública General de GNU (GPL), la Licencia MIT y la Licencia Apache. Cada una de estas licencias tiene sus propias reglas y restricciones.
  3. Licencia de Software Libre: Aunque a veces se usa de manera intercambiable con "código abierto", el software libre se refiere específicamente a software que se puede usar, modificar y distribuir de acuerdo con ciertas libertades. La Fundación del Software Libre (Free Software Foundation) promueve licencias como la GPL para garantizar estas libertades.

Hardawre

Es la parte física y tangible de un sistema, computadora o dispositivo electrónico.Son los componentes físicos que componen una máquina y que permiten su funcionamiento. Esto incluye componentes como la unidad central de procesamiento (CPU), la memoria RAM, el disco duro o unidad de estado sólido, la tarjeta madre, la tarjeta gráfica, el teclado, el ratón, la pantalla, los puertos USB y otros dispositivos de entrada y salida.


Es esencial para que un dispositivo pueda ejecutar programas y realizar tareas. Los programas de software, por otro lado, son las instrucciones y datos que se ejecutan en el hardware para realizar diversas funciones.


Software de sistema

La serie de programas preinstalados en el computador o sistema informático y que permiten interactuar con el Sistema Operativo (el software que rige el funcionamiento del sistema todo y garantiza su operatividad), para dar soporte a otros programas y garantizar el control digital del hardware. El software de base suele ser muy simple, inaccesible para la mayoría de los usuarios y administra el potencial físico del computador, conforme a las instrucciones del sistema operativo, con el que no debe confundírsele.

  • Sistema opertauvo: software que coordina y dirige todos los servicios y aplicaciones que utiliza el usuario en una computadora, por eso es el más importante y fundamental. Se trata de programas que permiten y regulan los aspectos más básicos del sistema.
  • Controladores: es un programa de software que indica al sistema operativo de su computadora cómo comunicarse con una determinada pieza de hardware.

Software de aplicación

programas de aplicaciones o en algunos casos aplicaciones, al conjunto de los programas informáticos generalmente instalados en el sistema por el usuario, y diseñados para llevar a cabo un objetivo determinado y concreto, de tipo lúdico, instrumental, comunicativo, informativo, etc. El software de aplicación normalmente es diseñado y comercializado aparte del software de sistema que viene incorporado de fábrica al computador, y su selección e instalación en el sistema es a total libre albedrío del usuario.

No tienen que ver con el funcionamiento del computador, lo instalamos en él para darle funciones determinadas como herramienta de trabajo (hoja de cálculo, procesador de palabras, programas de diseño gráfico, etc.), de ocio (videojuegos, reproductores de audio o video, etc.) o de información (enciclopedias digitales, navegador de Internet, etc.), entre otras.

  • Software de Productividad: Estos programas están diseñados para ayudar a los usuarios a realizar tareas productivas como procesadores de texto (Microsoft Word), hojas de cálculo (Microsoft Excel), software de presentación (Microsoft PowerPoint), herramientas de gestión de proyectos (Microsoft Project), y software de correo electrónico (Microsoft Outlook).
  • Software de Comunicación: Estas aplicaciones permiten la comunicación entre usuarios a través de diversas formas, como correo electrónico (Outlook, Gmail), mensajería instantánea (WhatsApp, Slack), videollamadas (Skype, Zoom) y redes sociales (Facebook, Twitter).
  • Software de Diseño y Creatividad: Este tipo de software se utiliza para crear contenido visual, gráfico o multimedia, como Adobe Photoshop (edición de imágenes), Adobe Illustrator (diseño gráfico), Adobe Premiere (edición de video) y software de modelado 3D (Blender, AutoCAD).
  • Software de Entretenimiento: Incluye aplicaciones diseñadas para el entretenimiento del usuario, como reproductores de música (iTunes, Spotify), reproductores de video (VLC, Netflix), videojuegos (Fortnite, Minecraft) y plataformas de streaming (Twitch, YouTube).

Software Malicioso

El software malicioso es cualquier software o aplicación móvil que se haya diseñado específicamente para perjudicar a los usuarios o dañar los ordenadores, los dispositivos móviles o el software que se ejecute en ellos. Infecta los sistemas informáticos, como ya sabemos. Su objetivo es infiltrarse en un sistema o dañarlo, sin el consentimiento del propietario. Las infecciones o ataques se realizan aprovechando las vulnerabilidades o agujeros de seguridad de los sistemas informáticos.

El malware causa daño en el software de aplicación y en el software de sistema, rara vez llega a causar problemas al hardawre, a continuación unos ejemplos de software malicioso:

  1. TROYANO:En informática, se denomina caballo de Troya, o troyano, a un software malicioso que se presenta al usuario como un programa aparentemente legítimo e inofensivo, pero que, al ejecutarlo, le brinda a un atacante acceso remoto al equipo infectado
  2. GUSANOS: Los gusanos informáticos se propagan de computadora a computadora, pero a diferencia de un virus, tiene la capacidad a propagarse sin la ayuda de una persona. Lo más peligroso de los worms o gusanos informáticos es su capacidad para replicarse en el sistema informático, por lo que una computadora podría enviar cientos o miles de copias de sí mismo, creando un efecto devastador a gran escala.
  3. ADWARE: Programa que automáticamente muestra u ofrece publicidad, ya sea incrustada en una página web mediante gráficos, carteles, ventanas flotantes, o durante la instalación de algún programa al usuario, con el fin de generar lucro a sus autores.
  4. RANSOMWARE: Restringe el acceso a determinadas partes o archivos del sistema infectado, y pide un rescate a cambio de quitar esta restricción.1 Algunos tipos de ransomware cifran los archivos del sistema operativo inutilizando el dispositivo y coaccionando al usuario a pagar el rescate.
  5. SPYWARE: Recopila información de un ordenador y después transmite esta información a una entidad externa sin el conocimiento o el consentimiento del propietario del ordenador.
